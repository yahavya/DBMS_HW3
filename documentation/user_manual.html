1. Application Overview

MovieFinder is a web application designed for movie enthusiasts who want to discover and explore films based on various criteria such as plot themes, actors, directors, genres, and ratings. The application provides powerful search and analysis tools to help users find their next favorite movie.

Key Capabilities

  • Search movies by plot keywords and themes
  • Find movies by title
  • Analyze genre trends and ratings
  • Discover actor collaborations
  • Explore director filmographies

2. Application Features

Feature 1 ‚Äî Plot Keyword Search

Search movies by keywords in the plot overview using MySQL FULLTEXT. Example inputs: "space exploration", "time travel", "artificial intelligence". Results are ranked by match relevance and then by rating.

Example: search for "space" returns Interstellar, WALL¬∑E, The Martian, etc.

Use Case: Discover themed movies (e.g., sci‚Äëfi about space).

Backend Query: query_1(connection, keywords)

Feature 2 ‚Äî Title Search

Full-text search over `movies.title` to find exact or partial title matches. Useful for quick lookups and fuzzy title matching.

Example: search for "Star" returns Star Wars, Star Trek, etc.

Backend Query: query_2(connection, search_term)

Feature 3 ‚Äî Genre Analytics

Aggregated metrics per genre: average rating, movie count, total and average revenue. Can filter to genres with at least N movies to avoid small-sample noise.

Use Case: Rank genres by average rating or revenue to identify top-performing categories.

Backend Query: query_3(connection, min_movies)

Feature 4 ‚Äî Actor Collaboration Finder

Find actors who co-appear frequently with a given actor. The query uses self-joins on `movie_cast` and groups by co-actor to count shared movies.

Use Case: List actors who worked with "Tom Hanks" at least 2 times.

Backend Query: query_4(connection, actor_name, min_collaborations)

Feature 5 ‚Äî Director Filmography Explorer

List a director's films meeting a minimum rating threshold and include top-billed cast. Uses joins between `movies`, `movie_crew` (filtered by job = 'Director') and a nested subquery on `movie_cast` to collect lead actors.

Use Case: Show Christopher Nolan's films rated ‚â• 7.0 with top 3 cast members.

Backend Query: query_5(connection, director_name, min_rating)
